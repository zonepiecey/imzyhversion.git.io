Designing a Fintech AI Experience

About

Corgi Labs is a YC-backed startup building a payments intelligence platform that helps businesses improve payment performance, reduce fraud, and grow revenue.

Project Overview

This project explored how one product could serve very different types of businesses, including ecommerce, subscription SaaS, AI SaaS, and financial institutions. Through research, I found that these customers were not solving the same payments problem, which led to a clearer product structure: one modular platform with different paths to value by segment.

Case Study Focus

My focus was on product definition and structure. I looked at how different business models defined success, where value showed up first, and how the platform should be organized so each segment could quickly understand where to start.
